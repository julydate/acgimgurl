随机二次元图片 API

目前系统共收录 700+ 张去重过后的图片，采用新浪图片，高速访问

API 基本调用格式（默认重定向到图片）：

api.php

参数（第一个为默认值）：

ssl = true / false / auto
size = large / mw1024 / mw690 / bmiddle / small / thumb180 / thumbnail / square
return = redirect / json / jsonpro / url / img

JSON调用格式

api.php?return=json

JSON数据

{
    "code":200, #图片状态码
    "acgurl":"https:\/\/ws1.sinaimg.cn\/large\/0072Vf1pgy1foxkfy08umj31kw0w0nng.jpg", #图片地址
    "width":"2048", #图片宽
    "height":"1152" #图片高
}
